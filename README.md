# Mendel

Mendel is a research and engineering effort to build a capable, lightweight model that balances on-device footprint and strong reasoning, planning, and tool-use abilities.

Goals:
- High capability density: pack reasoning, planning, and tool-use per parameter.
- Modularity: clear separation of concerns across repo modules.
- Measurable claims: every capability must have an associated eval.

Repository layout mirrors the project's levers and capabilities (see `MISSION.md` for full rationale).

Top-level modules include `architecture`, `distillation`, `quantization`, `retrieval`, `agent`, `control`, `eval`, `inference`, and `docs`.
